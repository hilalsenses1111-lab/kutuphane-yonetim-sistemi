# Kütüphane Yönetim Sistemi

Python ile geliştirilmiş, nesne yönelimli programlama (OOP) prensipleri kullanılarak hazırlanmış bir kütüphane yönetim sistemi projesidir. Proje, kitap ve kullanıcı işlemlerinin yönetilmesini, ödünç verme ve iade süreçlerinin takip edilmesini sağlamaktadır. Veriler JSON dosyasında saklanarak program kapatılıp açıldığında korunmaktadır.

## Özellikler

* Kitap ekleme
* Kitap listeleme
* Kitap silme
* Kitap güncelleme
* Kitap arama
* Kullanıcı ekleme
* Kullanıcı listeleme
* Kitap ödünç verme
* Kitap iade etme
* Ödünç kayıtlarını listeleme
* Verileri JSON dosyasında saklama
* Program yeniden açıldığında kayıtları yükleme
* Tkinter ile geliştirilmiş grafiksel kullanıcı arayüzü (GUI)

## Kullanılan Teknolojiler

* Python 3.12
* Tkinter
* Nesne Yönelimli Programlama (OOP)
* JSON Veri Saklama
* Git & GitHub

## Proje Dosyaları

* `main.py` → Konsol tabanlı uygulamanın başlangıç dosyası
* `arayuz.py` → Tkinter grafiksel kullanıcı arayüzü
* `kutuphane.py` → Kütüphane işlemlerinin yönetildiği sınıf
* `kitap.py` → Kitap sınıfı
* `kullanici.py` → Kullanıcı sınıfı
* `odunc.py` → Ödünç kayıtları sınıfı
* `data.json` → Verilerin saklandığı dosya

## Çalıştırma

### Konsol Arayüzü

```bash
python main.py
```

### Grafiksel Arayüz (GUI)

```bash
python arayuz.py
```
