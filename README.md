# Kütüphane Yönetim Sistemi

Python ile geliştirilmiş basit bir kütüphane yönetim sistemi.

## Özellikler

* Kitap ekleme
* Kitap listeleme
* Kitap silme
* Kitap güncelleme
* Kitap arama
* Kullanıcı ekleme
* Kullanıcı listeleme
* Kitap ödünç verme
* Kitap iade etme
* Ödünç kayıtlarını listeleme
* Verileri JSON dosyasında saklama
* Program yeniden açıldığında kayıtları yükleme

## Kullanılan Teknolojiler

* Python 3.12
* Nesne Yönelimli Programlama (OOP)
* JSON Veri Saklama

## Proje Dosyaları

* `main.py` : Programın çalıştırıldığı dosya
* `kutuphane.py` : Kütüphane işlemleri
* `kitap.py` : Kitap sınıfı
* `kullanici.py` : Kullanıcı sınıfı
* `odunc.py` : Ödünç kayıtları sınıfı
* `data.json` : Kayıtların saklandığı dosya

## Çalıştırma

```bash
python main.py
```
